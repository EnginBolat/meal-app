## Run Locally

Clone the project

```bash
  git clone https://github.com/EnginBolat/meal-app.git
```

Go to the project directory

```bash
  cd meal-app
```

Install dependencies

```bash
  flutter pub get
```

Run App

```bash
  flutter run
```
